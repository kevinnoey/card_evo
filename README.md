# 进化核心 (Evolution Core) MVP v1.0

卡牌进化类 Roguelike 原型验证项目，基于 Godot 4.6 Mono 构建。融合 Slay the Spire 式分支地图探索与卡牌进化机制。

## 项目概述

玩家扮演"渗透者"或"代码崩溃者"，在深网拓扑地图中通过分支路径探索，经历战斗、事件、商店和休息点，最终击败两章 Boss。核心机制围绕"短期生存 vs 长期投资"的策略抉择展开——每回合需权衡打出卡牌与注入 EP 进化卡牌。

## 运行环境

- **引擎**: Godot 4.6（Forward+ 渲染器）
- **分辨率**: 1920×1080
- **语言**: GDScript
- **物理引擎**: Jolt Physics
- **平台**: Windows（D3D12）/ Android

## 快速开始

1. 使用 Godot 4.6+ 打开 `project.godot`
2. 按 `F5` 运行项目
3. 在登录界面选择角色（渗透者 / 代码崩溃者）
4. 进入深网拓扑地图，选择路径节点开始探索

## 可玩角色

| 角色 | HP | EP/回合 | 核心机制 | 难度 |
|------|-----|---------|---------|------|
| 渗透者 | 70 | 3 | 穿甲 · 屏障 · 注入进化 | 中等 |
| 代码崩溃者 | 55 | 3 | 崩溃叠层 · 终结爆发 | 较高 |

- **渗透者**：攻防均衡，稳定成长。穿甲无视护甲，屏障吸收伤害，程序牌注入进化触发绽放。
- **代码崩溃者**：高风险高回报。叠崩溃层数放大攻击伤害，命中后清零。需要精准计算斩杀线。

## 核心系统

### EP 经济系统
- 每回合恢复 3 点 EP
- 打出卡牌消耗对应 EP（0-3）
- 右键注入任意卡牌消耗 1 EP，获得 2 点屏障
- 永久 EP 加成可通过事件/商店获得

### 储备池
- 回合结束时剩余 EP 自动注入储备池（上限 15）
- 达到 5/10/15 时各获得 1 层能量护盾（永久 -1 伤害）
- 满 15 点可免费升级手牌中一张卡牌

### 卡牌进化
- Lv.1 → Lv.2：注入 8 EP
- Lv.2 → Lv.3：注入 20 EP，触发"进化绽放"
- 进化绽放：卡牌费用变 0，可额外行动一次，全屏闪白特效
- 弃牌后注入进度完全保留（按进化家族追踪）
- **所有20张卡牌均可进化至 Lv.3**

### 战斗机制
- 手牌上限 5 张
- 屏障：临时HP，下回合消失，优先于格挡吸收伤害
- 格挡：本回合有效，下回合消失
- 能量护盾：永久减免，战斗结束后消失
- 起始牌库 10 张，抽牌堆耗尽时弃牌堆洗入

### 数据碎片
- 战斗胜利获得数据碎片（货币）
- 用于商店购买物品和事件选择
- 支持 HP 恢复、卡牌获取、EP 加成等消费

## Roguelite 地图系统

### 深网拓扑图
- 每章 15 层分支路径 + Boss 层
- 节点类型：战斗（⚔️）、精英（⚡）、事件（❓）、休息（🏕️）、商店（🛒）、Boss（💀）
- 赛博朋克风格：霓虹发光连线、数据流动画、扫描线背景
- 已完成节点标绿，可用节点脉冲发光

### 章节结构
| 章节 | 名称 | 普通敌人 | 精英 | Boss |
|------|------|---------|------|------|
| 第一章 | 外围防线 | 防火墙哨兵、脉冲中继器、数据腐化体 | 加密守护者、虚空信标 | 湮灭协议·原型 |
| 第二章 | 深层网络 | 脉冲中继器·改、数据腐化体·改 | 加密守护者·改、矩阵哨卫 | 主控中枢·格式化巨兽 |

### 事件系统（7种）
| 事件 | 描述 |
|------|------|
| 废弃数据库 | 扫描数据(+8EP) 或 销毁终端(+20碎片) |
| 黑市商人 | 购买过载芯片/纳米血清 或 举报商人 |
| 故障克隆舱 | 复制卡牌/+4EP 或 砸毁设备 |
| 数据风暴 | 抵抗风暴(-10HP,+5储备) 或 关闭防护(-8HP,+30碎片) |
| 游荡 AI | 优化卡牌(移除) 或 学习算法(+8EP) |
| 废弃军械库 | 全部回收(+3EP/回合) 或 拆解改装(+1护盾) |
| 神秘祭坛 | 献祭HP换EP 或 放弃离开 |

### 商店（6种商品）
| 商品 | 价格 | 效果 |
|------|------|------|
| 进化模块 | 60 | 选择一张卡牌 +8 EP |
| 纳米修复 | 40 | 恢复 20 HP |
| 随机卡牌 | 50 | 获得随机 Lv.1 卡牌 |
| 移除卡牌 | 70 | 永久移除一张卡牌 |
| 能量饮料 | 30 | 下场战斗 +2 EP/回合 |
| 储备增压 | 45 | EP 储备池 +8 |

### 休息点
- 恢复 15 HP 或 随机卡牌 +4 EP

### 战后奖励
- 每场战斗胜利后从 3 个随机奖励中选择 1 个

### 核心突破（章节过渡）
击败第一章 Boss 后，从 3 个突破协议中选择 1 个：
- **超限跃迁**：手牌上限 +1，每场战斗首回合失去 5 HP
- **涌动核心**：储备池上限 → 20
- **基因飞升**：所有 Lv.1 卡牌直接升至 Lv.2
- **屏障转化**：回合结束时 50% 屏障转化为永久最大 HP
- **超频过载**：永久 +1 EP/回合，受到伤害 +1
