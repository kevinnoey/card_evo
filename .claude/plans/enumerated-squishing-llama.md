# 角色选择界面 — 边框设计增强

## Context
当前角色选择界面中，两张角色卡片各有一个左侧头像区域（440x626 面板，内嵌 400x550 头像图），但头像周围缺少精美的装饰边框。用户希望在保持现有卡片布局的基础上：
1. 加大头像的视觉占比（移除头像区域内的多余装饰元素）
2. 为每个角色头像添加赛博朋克霓虹风格的装饰边框

## 修改文件
`scripts/ui/login_screen.gd` — 仅修改 `_create_char_card()` 函数中的左侧头像面板部分（约 440-505 行）

## 实现方案

### 1. 调整左面板布局（加大头像）

**当前**：左面板 440x626 内包含头像(400x550) + "(缩小)"提示 + 分隔线 + 菱形图标行，显得零碎。

**改为**：
- 头像占满左面板：portrait 从 position(20,35) size(400,550) → position(10,10) size(420,606)
- 删除 "(缩小)" 提示标签（第 478-485 行逻辑）
- 删除头像下方的装饰分隔线（第 488-492 行逻辑）
- 删除菱形图标行（第 495-505 行逻辑）
- 删除 170x170 的 `portrait_border` ColorRect（第 471-475 行逻辑）

### 2. 添加赛博朋克霓虹边框效果

在头像上方叠加以下装饰元素（添加到 `left_panel` 子节点）：

#### a) 主线框（4条发光边线）
用 4 个 ColorRect 组成矩形框，颜色使用角色主题色（渗透者=#00F0FF，崩溃者=#FF3B8C），半透明（alpha 0.5）：
- 上边线：position(6,6) size(428,2)
- 下边线：position(6,618) size(428,2)
- 左边线：position(6,6) size(2,612)
- 右边线：position(432,6) size(2,612)

#### b) L 形金色角标（8条线段，四角各2条）
与外框风格一致，金色 `Color("#FFD700", 0.7)`，臂长 24px，粗细 3px：
- 左上角：横线(10,10,24,3) + 竖线(10,10,3,24)
- 右上角：横线(406,10,24,3) + 竖线(427,10,3,24)
- 左下角：横线(10,617,24,3) + 竖线(10,596,3,24)
- 右下角：横线(406,617,24,3) + 竖线(427,596,3,24)

#### c) 顶部/底部中央菱形装饰
- 顶部中央：◇ 图标 Label，position(195,-2) size(50,24)，角色主题色 alpha 0.6
- 底部中央：◇ 图标 Label，position(195,608) size(50,24)，同上

#### d) 扫描线叠加层
在头像上方覆盖微妙的 CRT 扫描线效果：
- 创建 Control 节点作为扫描线容器，设 mouse_filter = IGNORE
- 每隔 4px 画一条 1px 高的黑色半透明线（Color(0,0,0,0.06)），宽 420px

#### e) 边框呼吸脉冲动画
在 `_create_char_card` 末尾，为主框线添加循环亮度脉冲动画：
```gdscript
var frame_tween = create_tween().set_loops()
frame_tween.tween_property(main_frame_top, "modulate:a", 0.3, 2.0)
frame_tween.tween_property(main_frame_top, "modulate:a", 0.7, 2.0)
# 对4条边线同步应用（或用tween_parallel）
```

### 3. 代码结构

将边框构建逻辑提取为独立函数 `_add_portrait_border(parent: Control, char_color: Color, panel_size: Vector2)`，使代码更清晰，便于复用。
